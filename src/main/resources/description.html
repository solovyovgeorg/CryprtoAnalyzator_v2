Криптоанализатор шифра Цезаря

Программа предназначена для шифрования, дешифрования и криптоанализа текста, зашифрованного с использованием шифра Цезаря. Шифр Цезаря является одним из простейших подстановочных шифров, основанных на сдвиге символов алфавита на фиксированное количество позиций.

Основные возможности программы

  • Шифрование текста — преобразование исходного сообщения путём сдвига символов на заданный ключ.
  • Дешифрование текста — восстановление исходного текста при известном ключе сдвига.
  • Статистический анализ — вычисление ключа сдвига на основе сравнения частот вхождения символов в зашифрованном файле и образце.
  • Брутфорс — расшифровка текста путем перебора всех возможных ключей шифрования, сравнение с образцом орфографии\проверочного слова .
