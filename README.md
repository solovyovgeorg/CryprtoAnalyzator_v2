# Криптоанализатор Цезаря

Программа для шифрования, расшифровки и брутфорса текста с использованием шифра цезаря.

## Функции:
- Шифрование текста с заданным ключом.
- Дешифрование текста с известным ключом.
- Брутфорс (перебор всех возможных ключей).
- Статистический анализ текста для подбора ключа по образцу.
- Работа с файлами и текстовыми строками.
- Графический интерфейс (Java Swing).

#Структура приложения:
```
├─ controller/         Контроллер приложения
├─ model/              Логика шифрования
├─ view/               Графический интерфейс ()
├─ operations/         Интерфейс и список операций
├─ exceptions/         Исключения
├─ other/              Вспомогательные классы: файловый обработчик, валидатор, класс подсчета статистики
└─ App.java            Главный класс приложения

 Также для описание программы, используется подгружаемый файл формата html для отображения в GUI 
└─ src/main/resources/description.html
```
